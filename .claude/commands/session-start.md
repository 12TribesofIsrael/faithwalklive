# Session Start

Beginning-of-session briefing for Faith Walk Live. Run this when starting a new session.

Do ALL of the following:

## 1. Read Context
- Read `docs/session-log.md` for the last session's handoff
- Read the memory files referenced in MEMORY.md for current project state
- Check `git log --oneline -10` for recent commits
- Check `git status` for any uncommitted work

## 2. Briefing
Give Thomas a short briefing:
- What was shipped last session (1-2 sentences)
- Any pending action items he had from last time
- What's queued up to build today (from the Next Session list)
- Any blockers or questions before we start

Keep it tight. No fluff. Ready to work.
